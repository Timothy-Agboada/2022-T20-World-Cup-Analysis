# 2022-T20-World-Cup-Analysis
Repo for a model that analyzes and visualizes data from the T20 World Cup in 2022, from best players and most conducive stadiums.

# Overview
* This model provides visualizations for all the data extracted from the T20 World Cup in 2022.
* The outputs include:
    * Most number of matches won
    * Players with the highest score in the most number of matches
    * Best bowler in the most number of matches
    * Most teams won by batting first
    * Most teams decide to bat first
    * Best stadiums to bat first
    * Best stadium to bowl first
* The dataset used has 17 columns
* Libraries such as pandas and plotly were used

# Notes
* Although a specific data set is used for this model, it can be easily altered to cater for other datasets.
* Data was acquired from www.statso.io
